# Notebooks

This directory is for **exploration only** — not production code.

- `archive/` — frozen notebooks from prior work (e.g. the original course homework).
- `exploration/` — ad-hoc experiments. Anything that earns its keep should graduate into `arxiv_rag/`, `api/`, or `evals/`.

Production code lives in the packages at the repo root. Notebooks are scratchpads, not deliverables.
